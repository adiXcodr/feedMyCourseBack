## Technologies used
* ReactJS
* Redux
* Firebase


### Installing and running
* First install Yarn in your system from `https://classic.yarnpkg.com/en/docs/install`
* In root directory, Install the dependencies by `yarn`
* Run in browser with `yarn start`
